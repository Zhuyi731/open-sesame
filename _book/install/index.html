1. 本地使用

安装：

通过npm进行全局安装，安装一次即可在本地各处使用。

$ npm install sesame-http -g

更新至最新版本

$ npm update sesame-http -g

安装完毕后使用如下指令查看当前版本

$ sesame-http -v

开发调试，在项目目录上使用命令

$ sesame-http open
